# Clean Film Gallery

Add sample photos processed with the **Clean Film** picture style here.

## Image Requirements:
- Format: `.webp`, `.jpg`, `.jpeg`, or `.png`
- Recommended aspect ratio: 3:4 (portrait)
- These images will appear in the gallery modal when users click "Lihat Galeri Foto"

## Example filenames:
- `clean-film-1.webp`
- `clean-film-2.webp`
- `clean-film-3.webp`
