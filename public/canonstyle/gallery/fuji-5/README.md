# Fuji 5 Gallery

Add sample photos processed with the **Fuji 5** picture style here.

## Image Requirements:
- Format: `.webp`, `.jpg`, `.jpeg`, or `.png`
- Recommended aspect ratio: 3:4 (portrait)
- These images will appear in the gallery modal when users click "Lihat Galeri Foto"

## Example filenames:
- `fuji-5-1.webp`
- `fuji-5-2.webp`
- `fuji-5-3.webp`
