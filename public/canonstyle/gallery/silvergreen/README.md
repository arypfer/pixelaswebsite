# Silvergreen Gallery

Add sample photos processed with the **Silvergreen** picture style here.

## Image Requirements:
- Format: `.webp`, `.jpg`, `.jpeg`, or `.png`
- Recommended aspect ratio: 3:4 (portrait)
- These images will appear in the gallery modal when users click "Lihat Galeri Foto"

## Example filenames:
- `silvergreen-1.webp`
- `silvergreen-2.webp`
- `silvergreen-3.webp`
